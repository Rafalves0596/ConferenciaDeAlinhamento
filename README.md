# Conferencia De Alinhamento
 "A partir da vibração excessiva da impressora 3D, a mesa de impressão pode ficar desalinhada, o que gera desperdício de matéria-prima e defeitos nas peças produzidas. Para evitar isso, será instalado um sistema que verifica constantemente o alinhamento da mesa."
